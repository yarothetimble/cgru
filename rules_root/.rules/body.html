This is RULES root.

RULES is a Web engine to view and modify special folders structures designed for CG projects tracking.

This Web engine requires HTML5 technologies to work. So it is available in latest versions of Firefox, Chrome, Safari. Other browsers HTML5 support is not enough. But if you are reading this text - it works.

Documentation (not finished)

This online version designed to show a CG project tracking and to ask questions.

You can use tmp folder for tests. It will be cleared periodically.

This online version is limited as its server does not has Afanasy, your machine does not have its filesystem locally and your browser can't launch system commands.

  • Thumbnails are not generated and updated.
  • You can't "Open Folder" and other operations with files(folder).
  • Dailies generation and movies to sequence explosion is disabled.

Guest account limitations

  • No files ulpoad.
  • Player annotations will not be saved, but you can try to draw.
  • No news subscription. You can't subscribe on interested channels(locations) to receive changes. But if you specify email, you will receive notifications on questions you asked or answered.
  • Playlist is not editable.
